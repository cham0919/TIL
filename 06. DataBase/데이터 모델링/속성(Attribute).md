# 속성의 특징

- 속성은 업무에서 관리되는 정보이다.
- 속성은 하나의 값만 가진다.
- 주식별자에게 함수적으로 종속된다. 즉, 기본키가 변경되면 속성의 값도 변경된다는 것이다.


<br/>

# 속성의 종류

- 분해 여부에 따른 속성의 종류
  - 단일 속성
  - 복합 속성
  - 다중값 속성
  
- 특성에 따른 속성의 종류
  - 기본 속성
  - 설계 속성
  - 파생 속성
